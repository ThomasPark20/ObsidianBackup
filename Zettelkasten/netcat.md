# netcat

## Usage
Able to send [[UDP]] or [[TCP]] (default) to any port! can make connections too!

## Hacking
`-u` allows to send UDP instead of default TCP
`-l` listen mode
`-v` verbose
`-n` only numeric ip no dns using resolve hostname etc
`-p` specify port
